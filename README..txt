Piccolino's OLED library - this version uses local ram (1K) and is the fastest
performing (good for animations, VU-Meters, Spectrum analyzers etc). Please use
the Piccolino_OLED_SRAM library for everyday operation as it only uses about
128 bytes of local RAM (video buffering done in SRAM).
